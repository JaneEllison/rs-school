Ввод отрицательных значений осуществляется с помощью кнопки "+/-". Например, вводим число "3", потом нажимаем кнопку "+/-", число меняется на "-3".

Возведение одного числа (n) в степень другого (m), осуществляется следующим образом: первым вводим число, которое хотим возвести в степень, например "2", затем нажимаем на кнопку "xn", после чего вводим число, соответствующее спени, например "3", нажимаем кнопку "=", результат 2 в 3й степени "8".

Вычисление квадратного корня из числа осуществляется следующим образом: сначала вводим число, затем нажимем на кнопку "√x".
